Presentacion de HTML5

¿Que es?

HTML5 es la última versión del lenguaje de marcado de hipertexto (HTML), que se usa para definir la estructura y el contenido de una página web. Es un estándar que evolucionó para mejorar la experiencia del usuario, añadir funciones multimedia nativas y facilitar la creación de aplicaciones web.
